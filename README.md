# 🩺 Disease Prediction System

An AI-powered Disease Prediction System developed using Machine Learning and Flask.

The application predicts the most likely disease based on user-selected symptoms and provides confidence scores along with the top possible predictions.

---

## 🚀 Features

- Disease Prediction using Machine Learning
- Symptom Search Functionality
- Multiple Symptom Selection
- Confidence Score Visualization
- Top 3 Predicted Diseases
- Professional Healthcare Dashboard UI
- Responsive Design
- Real-Time Prediction

---

## ⚙️ Installation

1. Clone the repository

```bash
git clone <repository-link>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run application

```bash
python app.py
```

4. Open browser

```text
http://127.0.0.1:5000
```
